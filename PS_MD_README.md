# 🚀 PS MD - WhatsApp Bot

> **Fully set up, ready to pair, and working perfectly!**

## ⚡ Quick Start (30 seconds)

```bash
npm start
```

Then open in browser:
```
http://localhost:9090/pairing
```

Scan the QR code with WhatsApp. Done! ✅

---

## 📁 What This Is

PS MD is a powerful **WhatsApp automation bot** that lets you:
- ✅ Control WhatsApp from commands
- ✅ Automate messaging
- ✅ Download videos/music
- ✅ Use AI features
- ✅ Manage groups
- ✅ And much more!

---

## 🌐 Web Interface

Beautiful web-based pairing interface with:
- 📱 **Live QR Code** - Scan with your phone
- 📋 **Pair Code** - 8-digit backup option
- 🔄 **Real-time Status** - See connection updates
- 📖 **Built-in Guide** - Step-by-step instructions

---

## 🎯 How It Works

```
1. npm start                    ← Start the bot
2. Open pairing website         ← Visit http://localhost:9090/pairing
3. Scan QR or enter code        ← Use WhatsApp phone
4. Confirm pairing              ← On your phone
5. Use commands in WhatsApp     ← Type .menu to see all
```

## ⚙️ Configuration

Edit `config.env`:

```env
OWNER_NUMBER=your_whatsapp_number    # ← Your number here!
BOT_NAME=PS MD
PREFIX=.
SEND_WELCOME=true
```

## 🆘 Quick Troubleshooting

| Issue | Fix |
|-------|-----|
| QR not showing | Refresh page (F5) |
| Can't connect | Make sure `npm start` is running |
| Wrong port | Edit `config.env` PORT setting |
| Need to repair | Delete `rm -rf auth_info_baileys/` |

---

## 📞 Key Points

1. ✅ Keep browser open until ✅ Connected appears
2. ✅ Use same WhatsApp account on phone
3. ✅ Need stable internet on both devices
4. ✅ Don't delete `auth_info_baileys/` folder
5. ✅ For help, check documentation files
